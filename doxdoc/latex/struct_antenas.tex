\doxysubsection{Detailed Description}
Estrutura que representa uma antena com coordenadas e frequência. 

Esta estrutura é utilizada para criar uma lista ligada de antenas, onde cada nó contém as coordenadas da antena (x, y), a sua frequência e um ponteiro para o próximo nó.
